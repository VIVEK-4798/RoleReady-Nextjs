## What Counts as Activity?

### ‚úÖ User Activities (6 types)
1. **skill_added** - Adding a new skill
2. **skill_updated** - Updating an existing skill
3. **resume_uploaded** - Uploading/updating resume
4. **readiness_calculated** - Calculating job readiness
5. **roadmap_step_completed** - Completing a learning step
6. **role_changed** - Changing target job role

### ‚úÖ Mentor Activities (4 types)
1. **skill_approved** - Approving a user's skill
2. **skill_rejected** - Rejecting a user's skill
3. **internship_added** - Posting a new internship
4. **job_added** - Posting a new job

### ‚ùå What's NOT Counted
- Logging in/out
- Viewing pages
- Searching
- Reading notifications
- Changing password
- Viewing jobs (only posting counts)

## Key Points

1. **Each activity = 1 contribution** (no weighting)
2. **Shows past 365 days** (rolling window)
3. **Grouped by date** (not by hour)
4. **Fire-and-forget logging** (never fails main operations)
5. **GitHub-style visualization** (weeks horizontal, days vertical)
